# TST files

TST files are currently suspected of containing clickable VR zones (doors, objects, etc.). What composes the "zones" is currently unknown.

| Address | Size (bytes) | Description               |
| ------- | ------------ | ------------------------- |
| 0x00    | 4            | Zone count (N)            |
| 0x08    | N\*16 bytes  | Zone data - Still unknown |
